# Worker SOP

1. Open the Worker page before travel and confirm the Job ID, address, access notes and assigned items.
2. Record **On the way**, **Arrived**, then **Started**.
3. Upload the required **Before** photo before work. Use the same angle for the **After** photo.
4. Report problems using Customer not home, Parts required, Safety concern or Scope question. Add enough detail for admin to decide.
5. Never perform unapproved add-on work. Wait for approval in the job scope.
6. Complete each checklist and upload evidence before selecting **Completed**.
7. Poor reception is acceptable: keep WhatsApp/media queued and confirm sync when reception returns.

For an immediate safety concern, stop work, make the area safe and call admin.
