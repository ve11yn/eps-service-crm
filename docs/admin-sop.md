# Admin SOP

## Daily opening

1. Review Dashboard exceptions, unresolved field alerts, overdue invoices and the Audit Log.
2. Qualify new leads and confirm the customer, property and requested work.
3. Review AI-generated text against the original evidence before confirming it.

## Quote to schedule

1. Build the quote from active catalogue items; verify quantity, price, discount and exclusions.
2. Create a revision for scope changes. Never overwrite an already-sent version.
3. After approval, confirm payment requirements, assign a team and set project/item dates.

## Field operations and closeout

1. Resolve worker issues with a written resolution.
2. Confirm required before/after evidence for every applicable work item.
3. Return incomplete work with a specific rework reason, or record customer sign-off and approve QA.
4. Review the generated invoice, record/verify payment, then complete the project.

## Controls

- Do not merge customers without checking contact and property histories.
- Do not approve Notion shutdown until every parity check and the parallel run are complete.
- Review Audit Log entries for discounts, status changes, assignments, QA and payments weekly.
